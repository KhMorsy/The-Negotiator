# Skills library

The [authoring guide](./SKILL_AUTHORING_GUIDE.md) is the source of truth for writing a negotiation skill. Authored catalog files live in `config/skills/catalog/`, grouped by category. Propose a skill by adding it to its category file, adding focused tests when its rule is new, and running the catalog validation suite.

Generated skills belong in `config/skills/generated/` and must satisfy the same validation rules.
